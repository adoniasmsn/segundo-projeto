Trabalhos realizados

Temos aqui o Primeiro projeto, em puro HTML, feito através do primeiro módulo do curso. O projeto proposto foi reproduzir o site de uma faculdade utilizando apenas códigos em HTML.

Tela inicial

Como não utilizamos CSS nesse projeto, ele foi construído dentro de uma tabela. Na página inicial utilizamos uma logo, fizemos um menu muito simples, colocamos uma imagem de capa e escrevemos um breve texto.

Quem somos

Aqui utilizamos o mesmo menu da página inicial, utilizamos textos gerados, uma lista ordenada e quebras de linha.

Contatos

Para finalizar foi utilizado comandos de formulários para criar a pagina de contatos. Nela utilizamos espaços para digitação e botões.
